HTML IMAGE SYNTAX

The Html <img> tag is used to embed an image in a web page.Images are not technically inserted into a web page, images are linked to web pages.The <img> tag creates a holding space for the reference image.
The <img> tag is empty, it contains attributes only, and does not have a closing tag
The <img> has two requred attributes
These are;

src
Specifies the path to the image
alt
Specifies the alternate text for the image

STYNTAX

<img src="url" alt="alternate text">

The src attribute

The required src attribute specifies the path (URL) to the image

NOTE:When a web page loads , it is the browser, at that moment, that gets the image from a web server and insert it into the page.Therefore, make sure that the image actually stays in the same sport in relation to the web page, otherwise your visitors will get a blocken link icon.
The broken link icon and the alt text are shown if the browser does not find the image

The alt attribute

The required alt attribute provides an alternate text for the image, if the user for some reasons cannot view the (because of slow connections, an error in the src attribute, or if the user uses a screen reader). The value of the alt attribute should describe the image.
If the browser cannot find the image, it will display the value of the alt attribute

HTML IMAGE MAPS

With HTML image maps, you can create clickable area in the image

Image maps

The Html <map> tag defines an image map.An image map is an image with clickable areas.The areas are defines with one or more <area> tags.

How does this works

The idea behind an image with map is that you should be able to perform defferent actions depending on where in the image you click.
To create an image map, you need an image, and some HTML codes that describes the clickable areas

The image

The image is inserted using the <img> tag. The only difference is that you add a usemap attribute
Here is an Example

<img src="C:\Users\KOINANGE\Documents\Ian's coding\html\My images\FB_IMG_16577352368899486.jpg" alt="Lips" style="width:250px;height:300px;" usemap="#Lips">

The usemap value starts with a hash tag # followed by the name of the image map, and is used to create a relatioship between the image and the image map

Create an Image map

Then create a <map> element.
The <map> element is used to create an image map and is linked to the name of the image by using the required name attribute

Here is an example

<map name="Lips">

The name attribute must have the same value as the <img> usemap attribute

The area

Then add a clickable area
A clickable area is defined by an <area> element.

Attributes present in the clickable area

1. Shape

You must include shape of the clickable area, and you can choose one of these values

  1. rect- defines a rectangular region
  2. circle- defines a circular region
  3. poly- defines an polygonal region
  4. default- defines the entire region
  1. Shape="rect"
    The coordinates for shape="rect" come in pairs, one for the x-axis and the other for the y-axis.

    So , the coodinates 34,44is located 34 pixels from the left of margin and 44 pixels from the top:

    The coordinates 270,350 is located 270 pixels from the left margin and 350 pixels from the top
  2. Shape="circle"
    To add a circle clickable area, first locate the coordinates of the center of the circle, Then specify the radius of the circle
  3. Shape="poly"
    This shape contains several coordinate points which creates a shape formed with straight line

    This can be used to create any shape
    Like maybe croissant shape

3. The Href attribute

The href gives the link of the path to be followed after clicking on the clickable area

HTML BACKGROUND IMAGES

In this chapter, I Ian Muindi will deal with background images

A background image can be specified for almost all the HTML element

Background images on a html element

To add a HTML image on an element, you add the style attribute and the background-image property

Here cames the example

<p style="background-image:url(img_girl.jpg);">

You can also specify the background image in the <head> selection

Here cames the example


 <head>
 <style>
   p {
  background-image:url(img_girl.jpg);
}
 </style>
 </head>

Background images add very important features and appearance as well.
